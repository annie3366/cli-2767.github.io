Homework #4 - me.json

This .json file will contain information about you. Read carefully to make sure content is structured uniformly. The top level will contain 4 keys:

  • name: An array with strings. Put as many names as you like, in the order you prefer. Mine looks like ["Matt", "Jacobson].
  • home: An object with 2 keys, latitude and longitude, each value is a number. This is the location you come from or where were you born. Whatever you call your first home.
  • url: A string. A url to some kind of website of yours (a portfolio, if you have one, a social media page, a github page etc).
  • favorites: An object with the keys and values
    • color: A string. A 6-digit hex value. What's your favorite color right now?
    • book: An object with keys "title" and "author", both values are strings.
    • tool: A string. Your favorite tool for creating (photoshop, camera, brush, etc).
    • temperature: A number. Your ideal outdoor temperature in celsius.


Helpful Hints

  • A .json file starts and ends with { }, and the inside reads just exactly like a Javascript object.
  • Use an online validator to make sure there are no errors in your syntax.

Submission
