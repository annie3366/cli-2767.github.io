Homework #3 - Recreating a Molnar

Create one webpage using HTML and JavaScript that recreates one of Vera Molnar's works


Requirements

  • Your page doesn't have to match the original work perfectly but should capture the "spirit" of the original.
  • Your pages body should only have a single svg element and javascript to append children to it.
  • Your page should be generative and should look different every time it is reloaded.
  • Don't worry about having your drawing be responsive to the window size.

Helpful Hints

  • Math.random() will be VERY useful - remember it returns a value between 0 and 1.
  • For loops are your best friend for this stuff.

Submission

You will turn in your homework by submitting a pull request to our class github repository and adding your index.html file to the folder corresponding to your NetID under /homework/3/ (i.e. cli-2767.github.io/homework/3/jacobsom/). Alternatively you can email me your html code and I will add it to the repo.
